# VisionUtils package

This approach is based on the Monocular Visual Odometry code provided in: https://github.com/avisingh599/mono-vo using the given approach as a feature detection and extraction template.

Requirements:
    OpenCV 3.X
    Qt5 for standalone


Install:
    cd <folder_path>
    mkdir build
    cd build
    cmake ..
    make
    sudo make install
